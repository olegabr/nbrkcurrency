== Description ==

* Show the exchange rate Central Bank of Russian Federation.
* Курс валют ЦБ РФ на текущий день c динамикой изменения курса RUR, RUB, EUR, USD.

**Supported Languages:**

* RU Russian (default)

== Installation ==

1. Unzip archive to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To add the CbrCurrency widget to the sidebar go to 'Appearance->Widgets', and add the CbrCurrency to your blog.

* Для включения виджета, после активации плагина переходим в:
* Внешний вид - 
* Виджеты - 
* Перетаскиваем виджет Cbr Currency на панель справа
